Class AuiStorySelection

source code

The Audio User Interface for the story selection frame. It is active when said frame is in 'locked' mode. Enables students to navigate and access functionality of their StoryBook.

deleteStory(self)

Called when delete key is released. If previous key pressed was also the delete key, deletes the current story. Otherwise, requests confirmation for delete.
