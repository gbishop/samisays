Class AuiStoryCreation

source code

Captures keys and gives audio cues to control the creation of a story. Allows user to create, navigate, and delete the clips of a story. Also enables user to enter the module for inserting sound effects. Exiting this module returns the user to managing his or her stories.

Method Details [hide private]

loadFullStory(self)

source code 
Plays wait sound while full story is loaded. Initially, only title was loaded for story selection.

setInstructions(self)

source code 
Sets current instructions based on state of the story. These will be spoken when user presses help key and displayed on the visualizer gui.

onKeyDown(self, event)

source code 
Handles event when a key is pressed. Keys track of first key to be held down and ignores all others until released. If key is record button, recording is started.

getHelp(self)

source code 
Called when help key is released. Reads the current instructions to user.

recordingFinished(self)

source code 
Ends recording of clip and crops the beginning so that the click will not affect normalization. If clip is longer than crop, inserts it into the story and replays it. Otherwise, notifies the user. If story was previously empty (this is the title) post-title instructions are appended to the endof the replayed clip.

playbackStory(self)

source code 
Starts a thread to play back the entire story in order. If interrupted by a valid key, thread returns and current clip is the last to begin being played. A separate thread is is used to allow this functionality.

insertSound(self)

source code 
Called when insert sound key is released. Passes control to the AuiInsertSound class to allow user to insert a sound effect.

insertBreak(self)

source code 
If in teacher mode, a break is inserted in the template. When the template is assigned to a student, clips between breaks will be combined into a single undeletable clip.

deleteClip(self)

source code 
Called when delete key is released. If clip is locked, notifies user. If previous key pressed was also the delete key, deletes the current clip. Otherwise, requests confirmation for delete. If current clip is title, notifies user that a new title must be recorded before anything else can be done.

quit(self)

source code 
Called when quit key is pressed. If last key pressed was also quit key, plays wait sound until all pickling threads have exited then returns to story selection. Otherwise requests confirmation.
